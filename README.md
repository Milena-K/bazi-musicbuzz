### To start the app
just run npm start

### Paths available
- "/" welcome page
- "/login" login page
  - "/login/artist"
  - "/login/listener"
- "/register" register page
  - "/register/artist"
  - "/register/listener"
- "/profile" profile page
- "/search"
- "/playlist/:playlistName"
- "/upload/song"
- "/upload/episode"
- "/create/podcast"
- "/create/album"
